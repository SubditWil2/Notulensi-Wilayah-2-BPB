I. Modul Notulensi Rapat

01

Metadata & OCR AUTO-FILL

Menentukan folder penyimpanan berkas secara otomatis berdasarkan data undangan.

  • Upload file Surat Undangan (PDF/Gambar).
  • Klik tombol 🔍 Baca Surat Otomatis. AI akan mengisi kolom Judul, Instansi, dan Tanggal secara otomatis.
02

Otomasi via Transkrip GEMINI AI

Merangkum rekaman rapat menjadi draf notulensi resmi tanpa mengetik ulang.

  • Tempelkan teks transkrip (Zoom/Teams) ke kolom Kirim via AI.
  • AI akan memilah dan merangkum Pembahasan serta Kesimpulan ke dalam draf Word.
03

Finalisasi Draf & WA Share

Editing detail draf dan pendistribusian draf kepada tim/pimpinan.

  • Klik tombol 📄 Buka File Notulensi (Word) untuk editing di cloud OneDrive.
  • Gunakan tombol Salin Teks WA untuk mengirim link draf secara resmi melalui WhatsApp.
📄

II. Modul Nota Dinas & Surat

01

Pembuatan Draf Berjenjang

Menentukan hierarki pejabat penandatangan untuk proses administrasi.

  • Pilih Target Penandatangan (Kasubdit s/d Menteri).
  • Sistem akan menyiapkan folder kegiatan dan draf Word sesuai template resmi.
02

Monitoring Status

Memantau progres draf dari tahap pembuatan hingga tanda tangan pimpinan.

  • Dokumen yang baru dibuat akan berstatus ⏳ Proses.
  • Gunakan Link draf Word di menu Rekap untuk konsultasi berjenjang.
⚙️

III. Finalisasi Dokumen (Update Data)

Digunakan untuk mengubah status draf dari ⏳ Proses menjadi ✅ Final.

01

Akses Kelola Berkas

  • Pada tabel Rekap, klik tombol aksi ⁝ (Titik Tiga) di baris dokumen Anda.
  • Pilih menu ⚙️ Kelola Berkas.
02

Upload Scan & Input Nomor

  • Upload file Scan PDF yang sudah ditandatangani pimpinan.
  • Input Nomor Surat Resmi dan pilih Tanggal Dokumen Final.
  • Status akan otomatis berubah menjadi "Final" setelah data disimpan.

📂 IV. Fitur: Arsipkan Berkas Eksternal

Digunakan khusus untuk menginput dokumen yang dibuat manual (di luar sistem) atau berkas lama.

  1. Buka menu Rekap ND & Surat.
  2. Di pojok kanan atas, klik tombol hijau 📂 Arsipkan Berkas.
  3. Pada pilihan mode di dalam modal, pilih "Arsipkan Baru".
  4. Input Provinsi, Nama Kegiatan, dan Perihal secara manual, lalu upload file scan PDF final beserta Nomor Surat resminya.
  5. Klik Simpan. Data akan langsung terdata sebagai berkas ✅ FINAL.

💡 Tips: Jika data master tidak muncul, silakan tekan F5 (Refresh) atau klik Batal.
